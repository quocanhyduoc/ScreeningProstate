## Cách sử dụng Docker 
Các lệnh kiểm tra và quản lý Docker hữu ích trên VPS:
Xem danh sách các Container đang chạy và cổng tương ứng:

```bash
docker compose ps
```

(Sẽ thấy cả 4 container: db, backend, frontend, và db-admin đều đang ở trạng thái Up)

Xem Log (lịch sử hoạt động/lỗi) của Backend và Frontend:

```bash
# Xem log của Backend (FastAPI)
docker compose logs -f backend
# Xem log của Frontend (Next.js)
docker compose logs -f frontend
```

Dừng toàn bộ hệ thống Docker:

```bash
docker compose down
```

Khởi động lại toàn bộ hệ thống Docker:

```bash
docker compose restart
```

Tải lại mã nguồn và rebuild lại toàn bộ hệ thống (dùng khi cập nhật code):

```bash
docker compose down
docker compose up --build -d
```

*Lưu ý quan trọng:* Hãy sử dụng lệnh `docker compose` (có dấu cách, không dùng dấu gạch ngang `-`) để tương thích với Docker Engine thế hệ mới và tránh lỗi Python traceback `KeyError: ContainerConfig`.
